## List all tags
```
<?php
	$tags = getTags();

	foreach ($tags as $tag) {
		echo 'Tag name: '	. $tag->name();
		echo 'Tag key: ' 	. $tag->key();
		echo 'Tag link: ' 	. $tag->permalink();
		echo 'Tag amount of pages: ' . count($tag->pages());
	}
?>
```
